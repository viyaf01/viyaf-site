Giving & Partnership Framework

捐赠与合作体系

从首次支持到家族传承——四条参与路径，一套可验证的交付与透明机制。本页所列门槛与权益为当期版本口径，可能随汇率、政策、场地与制作成本、项目预算等因素动态调整；最终条款以双方签署的《捐赠合约》《Impact Agreement》《Donor Benefits Matrix（当期版）》为准。

原则 01

纯公益属性

捐赠为自愿、无偿且不可撤回；不构成投资回报、利润分享、股权债权或治理权利。所列权益仅限艺术、教育与文化层面。

原则 02

用途清晰

捐赠主要用于青年艺术家培养、舞台制作与公益项目执行；定向项目/个人支持将在合规前提下写入合同并可追溯归档。

原则 03

交付可验证

节目单、影像档案、出版/发行编号、导师评估、捐赠收据与年度报告——每一份支持都有据可查。

Levels & Circles

参与路径与等级

一、Harmony Friends（入门参与）

面向首次支持者与普通捐赠人。Benefactor 可选八组 Ambassador 荣誉称号之一（爱心 / 爱乐 / 幸福 / 快乐 / 健康 / 和平 / 光明 / 慈善），亦可选择不使用称号或匿名。

级别年度捐赠礼遇（当期口径）
Harmony Friend≥ €100电子证书 · 月度简报
Harmony Patron≥ €1,000纸质证书 · 银质胸针 · 官方鸣谢（可匿名）
Harmony Benefactor≥ €10,000证书 · 银徽章 · 晚宴/私享沙龙 ×1 · 可选 Ambassador 荣誉称号

二、Rising Virtuosi（荣誉型 · 一次性捐赠）

适用于希望支持青年艺术家舞台成长与作品制作的捐赠人与家庭。本项并非购买演出机会，而是对专业评估、导师反馈、舞台实践、制作支持与项目归档的公益支持。本项为荣誉型（Honorary）；Fellow / Laureate 由艺术委员会与评审体系产生。捐赠人可提议受益人，最终安排须经艺术委员会评估、排期确认、未成年人保护审查、场地条件与版权授权确认。

为确保专业与公信力，所有「舞台实践机会」均需满足项目条件与专业评估；具体场次、曲目、阵容、场地规模与制作标准以当期方案为准。

三、爱乐贵族 Philharmonic Nobility（六级）

Three Separate Channels

捐赠 · 赞助 · 项目服务费——三条独立通道

三条通道分别使用独立合同、独立账户与独立会计口径，相互分离、各自披露。

捐赠 Donation / Spende

主体：VIYAF Privatstiftung（奥地利基金会）

文件：《捐赠合约》《Impact Agreement》。支持青年艺术、奖学金、大师班与公益项目；无投资回报或收益权利；票务、晚宴等对价性内容按公允价值分离处理，开具奥地利官方捐赠收据。

赞助 Sponsoring

主体：Meta Management GmbH（奥地利）

文件：《赞助协议》/《商业合作协议》。品牌露出、联合传播、客户活动与 ESG/CSR 内容；商业发票结算，不开具捐赠收据。

项目服务费 Program Fee

中国境内接收主体：深圳市人人爱乐文化科技有限公司

文件：《项目参与协议》。学生参与、大师班、旅行、教学排练与比赛训练等，属商业服务费，非 VIYAF 捐赠。

Giving Process

捐赠流程（七步）

意向沟通

确认支持方向：年度基金 / 项目基金 / 专项支持（Email / 线上 / 现场会议）。

路径确认

捐赠 / 赞助 / 项目服务费 / 专项合作——四条路径，合同与账目各自独立。

KYC / AML 尽调

身份与资金来源核验，通常 7–10 日（视材料完整度）。

签署文件

《捐赠合约》《Impact Agreement》及《Donor Benefits Matrix》等。

汇款确认

经官方邮箱二次确认账户后汇款；绝不通过聊天软件变更银行信息。到账后开具奥地利官方捐赠收据。

执行与归档

项目执行、节目单 / 影像 / 出版编号 / 导师评估 / 归档编号与简报。

年度复盘

项目报告 · 年度披露；续约 / 升级与下一年度支持方向。

Cross-Border

跨境汇款

支持对公/对私路径；建议配合专业顾问完成合规方案。VIYAF 提供标准账户信息与对账规则，全程可追溯。

税务处理

FAQ

常见问答

捐赠可以抵税吗？
取决于您的税务居民身份、所在司法辖区与受赠机构当期状态，请务必与您的税务顾问确认。VIYAF 可提供奥地利官方捐赠收据与相关合规文件。
支持可以定向到特定项目或方向吗？
可以。定向用途（如奖学金 / 舞台 / 录音 / Academy）将在合规基础上写入《捐赠合约》与《Impact Agreement》，形成可追溯的档案与交付记录。
为什么需要 KYC / AML？
作为跨境公益机构，VIYAF 依法进行必要的身份与资金来源核验，以保护基金会长期稳健与公信力——这同样是对每一位捐赠人声誉的保护。
涉及影像与未成年人如何处理？
影像采集、存储与使用遵循 GDPR、儿童保护与授权程序；公开范围以当期书面授权为准，并设有撤回与审批机制。
提名青年艺术家 / 受益人有什么规则？
达到相应等级的捐赠人可提名，须通过艺术委员会评审与排期安排；详情以当期《Donor Benefits Matrix》与项目计划为准。
跨境汇款如何操作？
支持对公与对私路径；建议配合专业顾问完成合规方案。VIYAF 提供标准账户信息与对账规则，汇款前请经官方邮箱二次确认。
企业适合捐赠还是赞助？
不涉及品牌露出、票务招待、logo 使用、传播权益等对价性内容时，可走捐赠通道；涉及品牌合作、客户活动或 ESG/CSR 内容时，应通过 Meta Management GmbH 签署商业合同，采用赞助或商业合作方式。
我的支持最终会留下什么？
VIYAF 致力于把支持转化为可验证成果：节目单、官方档案、影像记录、导师评估、出版/发行编号、归档编号与年度报告。重要的不是一场活动，而是可以被长期记录、复核与引用的成长档案。
