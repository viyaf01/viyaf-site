Giving & Partnership Framework

捐赠与合作体系

VIYAF 欢迎审慎、长期的支持，用于青年艺术家与公益项目。本页说明公开支持层级、捐赠与商业通道的严格分离、捐赠流程与账户安全规则。最终条款以双方签署的《捐赠合约》及相关文件为准。

原则 01

纯公益属性

捐赠为自愿、无偿且不可撤回；不构成投资回报、利润分享、股权债权或治理权利。所列权益仅限艺术、教育与文化层面。

原则 02

用途清晰

捐赠主要用于青年艺术家培养、舞台制作与公益项目执行；定向项目/个人支持将在合规前提下写入合同并可追溯归档。

原则 03

交付可验证

节目单、影像档案、出版/发行编号、导师评估、书面收款确认与年度报告——每一份支持都有据可查。

Levels & Circles

公开支持层级

各层级支持均用于经独立遴选的青年艺术家与公益项目。鸣谢保持克制；详细项目资料在正式沟通与尽调阶段提供。

公开层级建议金额公开回报
Supporter€1,000+年度影响力简报 · 可匿名或公开鸣谢
Patron€10,000+项目更新 · 年度成果沟通 · 适度鸣谢
Project Partner€50,000+专项预算与成果报告 · 冠名须另行合规审查
Legacy Partner私下沟通多年期支持或长期资本 · 仅在尽调后定制安排

金额为建议口径；详细项目预算与报告标准在尽调阶段向有意向的捐赠人提供。

捐赠与受益人遴选

捐赠人可建议支持领域或项目方向，但不购买录取、头衔、演出席位或其他个人结果。奖学金及青年艺术家支持由独立专业程序决定，并进行利益冲突审查。若付款人与拟受益人存在家庭或其他私人关联，相关安排将单独评估，并在适用时按商业服务处理。

Three Separate Channels

捐赠 · 赞助 · 项目服务费——三条独立通道

三条通道分别使用独立合同、独立账户与独立会计口径，相互分离、各自披露。

捐赠 Donation / Spende

主体：VIYAF Privatstiftung（奥地利基金会）

文件：《捐赠合约》《Impact Agreement》。支持青年艺术、奖学金、大师班与公益项目；无投资回报或收益权利；票务、晚宴等对价性内容按公允价值分离处理。VIYAF 可就实际到账金额出具书面收款确认，该确认本身不代表可税前扣除。

赞助 Sponsoring

主体：Meta Management GmbH（奥地利）

文件：《赞助协议》/《商业合作协议》。品牌露出、联合传播、客户活动与 ESG/CSR 内容；商业发票结算，不出具捐赠收款确认。Meta Management GmbH 为独立商业主体，并非 VIYAF 的子公司。

项目服务费 Program Fee

中国境内接收主体：深圳市人人爱乐文化科技有限公司

文件：《项目参与协议》。学生参与、大师班、旅行、教学排练与比赛训练等，属商业服务费，非 VIYAF 捐赠。

VIYAF 仅在适用法律允许的范围内接受跨境支持。VIYAF 未授权任何中国内地个人或机构代收捐赠，亦不通过微信、支付宝或个人账户收款；VIYAF 不在中国内地进行募捐。

Giving Process

捐赠流程（七步）

意向沟通

确认支持方向：年度基金 / 项目基金 / 专项支持（Email / 线上 / 现场会议）。

路径确认

捐赠 / 赞助 / 项目服务费——三条通道，合同与账目各自独立。

KYC / AML 尽调

身份与资金来源核验，通常 7–10 日（视材料完整度）。

签署文件

《捐赠合约》《Impact Agreement》及相关文件。

汇款确认

经官方邮箱二次确认账户后汇款；绝不通过聊天软件变更银行信息。到账后出具书面收款确认。

执行与归档

项目执行、节目单 / 影像 / 出版编号 / 导师评估 / 归档编号与简报。

年度复盘

项目报告 · 年度披露；续约 / 升级与下一年度支持方向。

Cross-Border

跨境与税务

跨境支持

VIYAF 仅在适用法律允许的范围内接受跨境支持。VIYAF 未授权任何中国内地个人或机构代收捐赠，亦不通过微信、支付宝或个人账户收款。任何跨境款项须经捐赠协议、身份与资金来源核验、银行审查及必要的司法辖区合规确认后处理。

收款确认与税务

VIYAF 可就实际到账金额出具书面收款确认。该确认本身不代表捐赠可在奥地利、中国内地、香港或其他司法辖区税前扣除。税务可扣除性取决于捐赠人税务居民身份、适用法律以及 VIYAF 在捐赠当日的法定资格。

教育与升学

VIYAF 提供导师支持、作品集与舞台交付等「可验证材料」；院校录取由学校独立决定。

FAQ

常见问答

捐赠可以抵税吗？
VIYAF 可就实际到账金额出具书面收款确认。该确认本身不代表捐赠可在奥地利、中国内地、香港或其他司法辖区税前扣除；税务可扣除性取决于捐赠人税务居民身份、适用法律以及 VIYAF 在捐赠当日的法定资格，请务必与您的税务顾问确认。
支持可以定向到特定项目或方向吗？
可以。定向用途（如奖学金 / 舞台 / 录音 / Academy）将在合规基础上写入《捐赠合约》与《Impact Agreement》，形成可追溯的档案与交付记录。
为什么需要 KYC / AML？
作为跨境公益机构，VIYAF 依法进行必要的身份与资金来源核验，以保护基金会长期稳健与公信力——这同样是对每一位捐赠人声誉的保护。
涉及影像与未成年人如何处理？
影像采集、存储与使用遵循 GDPR、儿童保护与授权程序；公开范围以当期书面授权为准，并设有撤回与审批机制。
可以建议受益人吗？
捐赠人可建议支持领域或项目方向，但不购买录取、头衔、演出席位或其他个人结果。奖学金及青年艺术家支持由独立专业程序决定，并进行利益冲突审查；若付款人与拟受益人存在家庭或其他私人关联，相关安排将单独评估，并在适用时按商业服务处理。
跨境汇款如何操作？
仅在适用法律允许的范围内，并须经捐赠协议、身份与资金来源核验、银行审查及必要的司法辖区合规确认。VIYAF 未授权任何中国内地个人或机构代收捐赠；请与您自己的专业顾问配合完成合规安排。
企业适合捐赠还是赞助？
不涉及品牌露出、票务招待、logo 使用、传播权益等对价性内容时，可走捐赠通道；涉及品牌合作、客户活动或 ESG/CSR 内容时，应通过 Meta Management GmbH 签署商业合同，采用赞助或商业合作方式。
我的支持最终会留下什么？
VIYAF 致力于把支持转化为可验证成果：节目单、官方档案、影像记录、导师评估、出版/发行编号、归档编号与年度报告。重要的不是一场活动，而是可以被长期记录、复核与引用的成长档案。
